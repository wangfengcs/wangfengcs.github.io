Mathjax语法总结

发表于 2017-07-19 | 分类于 工具书 | 0 comments

hexo写markdown中_与*一样，表示斜体或粗体，但mathjax中_表示下标，因此：
　　mathjax中换行\，在markdown中写成\\
　　mathjax中下标_，在markdown改写成_
　　mathjax中下标{，在markdown改写成\{
　　mathjax中下标}，在markdown改写成\}
在markdown的表格中插入|，可以输入 &#124; 代替
